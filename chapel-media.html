Chapel Media Equipment Upgrade

Project summary

The current chapel media setup relies on old SD projectors and an analog control/splitter setup. This proposal moves the chapel to HD projectors with HDMI signal control and Cat6 cable runs, keeping the existing projector mounting approach while modernizing the actual display and signal path.

Why this is needed

Clearer worship media

HD projection improves readability for lyrics, scripture, sermon points, announcements, videos, and visual teaching aids.

Works with the space

TVs are not a good fit for the chapel layout. This keeps the existing projector-based approach while replacing the outdated equipment.

Cleaner signal path

HDMI-over-Cat6 distribution provides a modern control setup and cleaner cable runs from the media area to the projectors.

Grant form wording

Item/Project Requested: Chapel Media Equipment Upgrade

Description: Replace the chapel’s aging SD projector system and analog media distribution with an HD projector setup using HDMI-over-Cat6 signal distribution. The system will support worship services, scripture and lyric projection, sermons, announcements, children’s programs, and community presentations. Because TVs are not appropriate for the chapel space, this project keeps the existing projector-based approach while upgrading the projectors, signal control, and cabling for a clearer and more reliable presentation system.

Prepared for The Salvation Army Women’s Auxiliary Grant Request 2026.
